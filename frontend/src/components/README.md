# Chat App

A full-stack real-time chat application built with React, Node.js, Express, MongoDB, Socket.io, Zustand, and DaisyUI.

## Features

- User authentication (signup, login, logout)
- Real-time messaging with Socket.io
- Online users indicator
- Profile management (update profile, profile picture upload)
- Responsive UI with DaisyUI and Tailwind CSS
- Light and dark themes
- Secure JWT-based authentication with httpOnly cookies

## Tech Stack

- **Frontend:** React, Zustand, React Router, DaisyUI, Tailwind CSS, Axios, Socket.io-client
- **Backend:** Node.js, Express, MongoDB (Mongoose), Socket.io, bcrypt, dotenv, cookie-parser, Cloudinary (for image uploads)
- **Other:** Vite (for frontend build), JWT
